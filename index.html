Desafio de projeto - Criar uma página web utilizando as tags aprendidas

Ao longo do curso foi visto como montar a estrutura básica de uma página html para ser visualizada pelos navegadores de internet (browsers), foi visto também alguns atributos que as tags podem ter, esses atributos ajudam a deixar a página mais personalizada e deixando-as com uma aprência mais específica.

Tags de título e subtítulos - h1 até h6

O primeiro título da página deverá estar dentro da tag h1, o h1 terá a maior fonte entre os títulos, os seguintes subtítulos deverão estar dentro das tags h2, h3, h4, h5 e h6, quanto maior o número após o "h", menor será o tamanho da fonte do subtítulo.

Exemplos:

Tag de parágrafo - p

As tags "p" são usadas para inserir parágrafos e textos dentro delas.

Tag mark

A tag mark serve para deixar um trecho específico de um texto (ou o texto inteiro) com um fundo amarelo (padrão da tag mark), a cor de fundo pode ser personalizada com o atributo "background-color", vamos ver a seguir alguns exemplos.

texto 1.
texto 2.

  • No texto 2, usamos o atributo background-color para deixar o fundo verde-claro (style="background-color: lightgreen;"), o atributo foi utilizado dentro da própria tag (mark), mas veremos mais adiante como utilizar os atributos de tags dentro do CSS.

Tag small

A tag small serve para deixar um texto menor em relação ao tamnho original de um parágrafo por exemplo. Veja um exemplo a seguir:

Uma frase qualquer - Uma frase qualquer

Tag i

A tag i serve para deixar tudo o que estiver dentro da tag i em itálico.

Texto normal - Texto em itálico

Tag u

A tag u serve para deixar uma palavra ou texto com uma linha sublinhada.

Texto 1 - Texto 2

Tag strong

A tag strong serve para deixar em negrito tudo o que estiver dentro dela (tag stroong).

Texto 1 - Texto 2

Tag ol (ordered list)

A tag ol (lista ordenanda) lista ítens um abaixo do outro de forma numérica ordenada, dentro da tag ol deverá ter uma tag li (list item) para cada ítem númerico da lista ordenanda.

  1. Ítem I
  2. Ítem II
  3. Ítem III

tag ul (unordered list)

A tag ul (lista não ordenanda) funciona de forma semelhante a tag ol, mas em vez de números para cada ítem, a lista será mostrada por pontos, tags li também deverão estar dentro da tag ul.

  • Ítem I
  • Ítem II
  • Ítem III

Tag a (ancor)

A tag a (âncora) serve para colocarmos um link em um determinado texto, dentro da tag a devemos usar o atributo seguido do link, veja um exemplo a seguir: href="https://www.w3schools.com".

Visit W3Schools.com!

Obs: para abrir a página em outra aba, devemos usar o atributo target="_blank"

Tag hr (horizontal line)

A tag hr serve para colocarmos uma linha horizontal em um determinado lugar de uma página.

Tags sub e sup

As tags sub e sup servem para deixar os textos subescritos ou sobrescritos.

tag sub: 123

tag sup: 123

Tag blockquote

A tag blockquote serve para inserir citações em determinado texto, indicando que a citação é de outro autor.

Um texto para exemplo

"Uma citação de um determinado autor."

Tag font

A tag font não é mais usada no HTML5, usamos o font como atributo no CSS3.

Tag del

A tag del serve mostrar uma parte de um texto que foi "deletada", o que estiver dentro da tag del ficará marcada com um traço no meio da(s) palavra(s), veja o exemplo a seguir:

"Um trecho deste texto não foi excluído."

Tag abbr (abreviatura)

A tag abbr serve para abreviar tudo o que estiver dentro dela, exemplo:

The WHO was founded in 1948.
